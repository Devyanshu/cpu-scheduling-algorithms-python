# CPU Scheduling Algorithms in Python with GUI.

### Python program made during Hackathon at Bennett University.

### Algorithms included:
 - First come first serve
 - Round Robin
 - Shortest Job First
 - Shortest Remaining time first
 - Priority (Non-preemptive)
 - Priority (Preemptive)
